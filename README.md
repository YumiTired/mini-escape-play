# Flight Mini-Escape ✈️

A tiny interactive terminal puzzle.  
No installation, no source code, no setup required — just Docker.

## Requirements
- Docker Desktop (Windows / macOS)  
- Docker Engine (Linux)

If you don't have Docker installed yet, you can get it here:  
https://www.docker.com/get-started/

## How to Play
Open your terminal:

- **Windows:** PowerShell  
- **macOS:** Terminal  
- **Linux:** any shell

Then run:

```bash
docker run --rm -it yumitired/mini-escape:latest
